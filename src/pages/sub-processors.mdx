This page lists the Sub-processors used by FailWarn.com to provide the Services as defined in the Terms of Service.

### Google

- **Sign-In with Google**: Facilitates user authentication through Google accounts.

### GitHub

- **Sing-In with GitHub**: Facilitates user authentication through GitHub accounts.

### Plausible

- **Plausible Analytics**: Provides website analytics without tracking or storing personal data.
